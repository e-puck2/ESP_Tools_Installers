Steps to configure ESP32 tools:0)	First Git and other things must be present. Run the following command in a terminal window	"sudo apt-get install git wget make libncurses-dev flex bison gperf python python-serial"Run the Install ESP32 tools.sh or follow the steps below :1)	Open a terminal window and cd it into a folder named esp you created2)	git clone --recursive https://github.com/e-puck2/esp-idf.git3)	Download xtensa-esp32-elf 32bits or 64 bits depending on you OS		32bits : https://dl.espressif.com/dl/xtensa-esp32-elf-linux32-1.22.0-73-ge28a011-5.2.0.tar.gz	64bits : https://dl.espressif.com/dl/xtensa-esp32-elf-linux64-1.22.0-73-ge28a011-5.2.0.tar.gz	4) 	Unzip and copy it to the esp in order to have a xtensa-esp32-elf folder5)	Copy the script Run ESP32 tools.sh located in /tools/ to the esp folderSteps to use the esp32 environment :1)	Open a terminal window and execute Run ESP32 Tools.sh script2)	Cd it into your ESP32 project folder3)	You can now use the usual commands to compile, flash, etc**If a message « Could not open serial port /dev/tty* » appears when trying to program, you have to type the following command in the terminal to put your user into the dialout group.« sudo adduser $USER dialout »Then restart your linux session and it should works.
